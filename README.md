# SmartExpenseTracker
A simple Java console application built for the Programming in Java flipped course.

Helps users manage daily expenses by adding, viewing, categorizing, and deleting expense entries.

Built using core Java OOP concepts: classes, objects, constructors, methods, and ArrayList.

Contains three main classes:

     Main.java → handles menu and user interaction

     Expense.java → represents each expense record

     ExpenseManager.java → manages operations like add, view, delete, and calculate total

Follows a modular and clean structure for readability and maintainability.

Screenshots included in the /screenshots folder as execution proof for evaluation.

Demonstrates practical application of Java basics to solve a real-world problem simply and effectively.
